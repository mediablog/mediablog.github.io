程序使用

打开 config.toml 文件，修改配置，每一项都有一个开关，选择自己需要操作的选项。
将需要处理的视频全部放入video文件夹(支持多目录)，然后运行

下载的程序包中已填好样例配置，只需填入密钥即可双击启动，查看效果

解压文件夹后
mac 运行
cd mac 进入到mac 文件夹下
./vm

windows 运行
双击vm.exe 文件即可

文件夹中已经配置了示例配置和演示用的视频，直接运行即可查看效果
